戦車の移動

戦車を移動させます。(いわゆるラジコン操作です。)
3つのフラグによって、前進と、左右の向き変えが行えます。
前進は10m/s、左右の転回は90度/sの速度です。

引数
[in]isForward前進するか否か
[in]isLeft左に回頭するか否か
[in]isRight右に回頭するか否か
注意
isLeft と isRight が共に true の時は転回しませんが、エネルギーは消費します。

TankManager, TankMovementで実装されています。
